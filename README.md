## ⚠️ Important Notice

This is a **public collaboration repository** maintained by the Product Management and Design team at Sparquo on behalf of a client. It does **not** contain the full application source code.

This repository exists solely to facilitate structured collaboration between the **PM/Design team and the development team**. Selected files — such as product documentation, specifications, and design references — are shared here to streamline the handoff process.

---

## 📁 What's in this repository

| File / Folder | Purpose |
|---|---|
| `docs/` | Feature specifications |
| `design/` | Design references, component notes, and Figma links |
| `flows/` | User flows and interaction logic per role |
| `CHANGELOG.md` | Version history and release notes |

> Everything else lives in a **private PRD and Notion space** accessible only to the core development team.

---

## 🤝 Collaboration scope

This repository is intended for:

- **Product Manager → Dev:** Sharing feature specifications
- **Designer → Dev:** Sharing design tokens and component documentation
- **Dev → PM/Design:** Raising questions, flagging technical constraints, and confirming implementation details via Issues

If you are a developer working on this project and need access to the full codebase, contact the project lead directly.
